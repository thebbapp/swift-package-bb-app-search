# ``BbAppSearch``

Content search functionality and user interface.

## Overview

The BbAppSearch library provides comprehensive search capabilities for finding posts, comments, and other content. It includes the main search view interface, autocomplete suggestions with ``BbAppSearchAutocompleteView``, search content results with `BbAppSearchContentView`, the `BbAppSearchService` that handles search queries and result aggregation, search layout components, error handling, and intent destination integration for navigation to search results.
